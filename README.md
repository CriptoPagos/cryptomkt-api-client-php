Instalación
-----------

Directamente desde la terminal con:

        $ composer require sasco/cryptomkt-api-client

O editando el archivo *composer.json* y agregando:

        {
                "require": {
                         "sasco/cryptomkt-api-client": "1.*"
                }
        }
